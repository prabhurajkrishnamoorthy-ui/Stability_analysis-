# KTP Stability Study — Data Analysis

Accelerated stability study analysis for two pharmaceutical formulations (A014 vs B025), covering data cleaning, trend analysis, formulation comparison and degradation modelling.

## Requirements

```bash
pip install pandas numpy matplotlib seaborn scikit-learn scipy openpyxl
```

## How to run

1. Clone this repository
2. Place `KTP_task_datasets.xls` in the same folder as the notebook
3. Open `Assessment.ipynb` in Jupyter
4. Run all cells top to bottom

## What the notebook covers

- **Q1** - Key features of the dataset
- **Q2** - Trends and patterns in the data
- **Q3** - Factors influencing degradation
- **Q4** - Comparison of formulations A014 vs B025
- **Q5** - Exponential decay modelling
- **Q6** - Limitations and future study suggestions
